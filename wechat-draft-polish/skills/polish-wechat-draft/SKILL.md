# Polish WeChat Draft

## Overview

This skill helps polish existing WeChat article drafts using proven techniques from viral content analysis and authentic writing styles. It enhances titles, emotional expression, readability, and formatting while preserving the original meaning and structure.

**Important**: This skill requires an existing draft. It polishes and refines content but does not create articles from scratch.
